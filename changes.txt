HOW IT WORKS
================================================================================

1. USER SEES: Portfolio section with 14 project folders
   - Each folder shows a thumbnail image
   - Folder name and item count displayed

2. FILTER: User can filter by category
   - All Projects - shows all 14 folders
   - Steel Structures - shows Baumit, CimCoop, Portfolio, CV
   - EV Infrastructure - shows all MFG projects
   - Education - shows schools (Bishop David Brown, Castle View, etc.)

3. CLICK FOLDER: Opens folder contents
   - Breadcrumb shows: Back / Folder Name
   - Grid displays all images and PDFs in that folder
   - PDFs show thumbnail of first page (via PDF.js)

4. CLICK ITEM: Opens modal viewer
   - Images display full-size, centered
   - PDFs display in iframe with zoom controls
   - Zoom: +/- buttons, reset, download link

5. NAVIGATION:
   - Click "Back" or press Escape to return to folders
   - Press Escape to close modal

================================================================================
TECHNICAL DETAILS
================================================================================

PDF.js Library:
- CDN: https://cdnjs.cloudflare.com/ajax/libs/pdf.js/3.11.174/pdf.min.js
- Worker: https://cdnjs.cloudflare.com/ajax/libs/pdf.js/3.11.174/pdf.worker.min.js
- Renders first page of PDF to canvas for thumbnails

Folder Data Structure:
folderData = {
    'folder-id': {
        name: 'Display Name',
        category: 'steel|ev|education',
        items: [
            { type: 'image|pdf', src: 'path', title: 'Title', desc: 'Description' }
        ]
    }
}
